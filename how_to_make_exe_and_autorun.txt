## ✅ Step-by-step

```bash
pip install pyinstaller
```

---

### 🔹 Step 2: build exe

আপনার `app.py` যেখানে আছে, সেখানে CMD খুলে:

```bash
pyinstaller --onefile --add-data "templates;templates" --add-data "static;static" app.py
```

* `dist/` ফোল্ডারে `.exe` তৈরি হবে

### 🔹 Step 3: run

```bash
dist\app.exe
```

👉 Browser-এ খুলুন:

```
http://127.0.0.1:5000
```

---

## ⚠️ Important fix (Flask + PyInstaller)

আপনার `app.py`-তে নিচেরটা add করুন:

```python
import os, sys

def resource_path(relative_path):
    try:
        base_path = sys._MEIPASS
    except Exception:
        base_path = os.path.abspath(".")
    return os.path.join(base_path, relative_path)

app = Flask(__name__,
    template_folder=resource_path("templates"),
    static_folder=resource_path("static")
)
```

👉 না দিলে exe-তে HTML load হবে না

# ⚡ ২) Auto-start server (Windows boot এ auto run)

## ✅ Method 1: Startup folder (সবচেয়ে সহজ)

### 🔹 Step:

1. Press `Win + R`
2. লিখুন:

```
shell:startup
```

3. আপনার `.exe` ফাইলটা এখানে copy করুন

👉 এখন PC চালু হলেই app auto run হবে

---

## ✅ Method 2: Task Scheduler (professional way)

1. Open: **Windows Task Scheduler**
2. Create Task
3. Trigger:

   * “At log on”
4. Action:

   * Program/script → আপনার `.exe` path

👉 Done ✅

---

```python
import webbrowser

    app.run(debug=False)
```

# 🖥️ ২) System Tray App (control panel)

Python tray UI: **pystray**

## ✅ tray_app.py

```python
import threading
import subprocess
import webbrowser
import pystray
from PIL import Image

server_process = None

def start_server():
    global server_process
    if server_process is None:
        server_process = subprocess.Popen(["server.exe"])

def stop_server():
    global server_process
    if server_process:
        server_process.terminate()
        server_process = None

def open_app():
    webbrowser.open("http://127.0.0.1:5000")

def exit_app(icon, item):
    stop_server()
    icon.stop()

def setup(icon):
    start_server()

icon = pystray.Icon("MyApp")
icon.icon = Image.open("icon.png")

icon.menu = pystray.Menu(
    pystray.MenuItem("Open App", lambda: open_app()),
    pystray.MenuItem("Start Server", lambda: start_server()),
    pystray.MenuItem("Stop Server", lambda: stop_server()),
    pystray.MenuItem("Exit", exit_app)
)

icon.run(setup)
```

# 🚀 ১) Final `server.py` (background run)

```python
from waitress import serve
from app import app
import webbrowser
import threading

def open_browser():
    webbrowser.open("http://127.0.0.1:5000")

if __name__ == "__main__":
    threading.Timer(2, open_browser).start()
    serve(app, host="127.0.0.1", port=5000)
```
